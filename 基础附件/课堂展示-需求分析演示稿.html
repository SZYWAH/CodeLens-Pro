Requirement Analysis

CodeLens Pro

一个帮助学生读懂代码逻辑、追问 AI 报告、沉淀知识，并通过 VS Code 插件参与项目协作的智能编程学习工具。

Background

时代现状

AI 让很多人不再需要手敲代码，但如果看不懂代码逻辑、改不了现有项目，竞争力还是不够。

01
会生成，不会理解代码是跑出来了，但不知道每一段为什么这样写，也不知道它具体在做什么。
02
会复制，不会修改遇到已有项目时，只能照着 AI 提示改，不能自己定位问题和调整逻辑。
03
会提问，不会复盘问完 AI 就结束了，没有把理解、知识点和经验沉淀下来。
Need

核心需求

真正有竞争力的不是“会不会让 AI 写”，而是“能不能读懂、改懂、讲清楚代码”。

Need 01读懂代码本质至少要知道每行代码的作用、函数之间的关系，以及程序整体的运行流程。
Need 02看懂后能追问当报告解释不清时，可以继续追问，直到把逻辑真正弄明白。
Need 03理解后可沉淀把关键知识保存成卡片、资料和笔记，后面复习时能快速回看。
Need 04学习过程可展示用报告、日志和统计数据说明自己不是“只会生成”，而是“真正学会了”。
Application

功能简介

工具围绕“看懂代码、持续追问、沉淀知识、协作修改”四个环节设计。

01代码分析输入代码后先做静态分析，再生成流式报告，帮助快速理解逻辑和风险。
02报告追问围绕当前报告继续提问，补齐看不懂的地方，直到理解为止。
03知识沉淀把关键结论保存成知识卡片，再生成学习资料、笔记和复习状态。
04Agent 插件通过 VS Code 插件同步项目上下文，生成修改计划，用户确认后再执行。
完整性有前端、后端、数据库和 AI 接口，能形成完整闭环。
智能性AI 参与分析、问答、卡片和日志生成，不只是单次回答。
美观度提供亮色与暗色两套主题，兼顾课堂投影、长时间阅读和不同使用偏好。
